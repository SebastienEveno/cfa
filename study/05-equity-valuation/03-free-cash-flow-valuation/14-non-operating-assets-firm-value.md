## Summary: Non-Operating Assets and Firm Value (CFA Level II)

---

### Core Concept

FCFF/FCFE valuation captures the value of **operating assets only** — those generating operating cash flows. Any **non-operating assets** must be valued **separately** and **added** to the operating value.

$$\boxed{\text{Total firm value} = \text{Value of operating assets (FCFF/FCFE-based)} + \text{Value of non-operating assets}}$$

---

### What Counts as a Non-Operating Asset?

| Category | Examples |
|---------|---------|
| **Excess cash** | Cash beyond what's needed to run operations |
| **Excess marketable securities** | Investment portfolios beyond working capital needs |
| **Investment real estate** | Land/property not used in operations |
| **Non-controlling equity stakes** | Investments in stocks/bonds of other companies that are financial investments, not operating subsidiaries |

---

### General Rule

> **Any asset excluded from the cash flow projection must be added back separately at its estimated value.**

This is necessary because:
- FCF projections only capture cash flows **generated by operating assets**
- Non-operating assets don't contribute to (or are excluded from) those projected cash flows
- Without this addition, the valuation would **understate** total firm/equity value

---

### Valuation Basis: Market Value, Not Book Value

**Key adjustment**: Non-operating financial investments (stocks, bonds, securities held for investment) reported at **book value** under accounting conventions should be **revalued to current market value** for the purposes of this addition.

> Book value may significantly understate (or overstate) the true economic value of these holdings — always use **current market prices** when available.

---

### Practical Application

```
Step 1: Identify all non-operating assets on the balance sheet
Step 2: Determine current market value for each
         (revalue book-value items to market value)
Step 3: Estimate operating asset value via FCFF/FCFE model
         (cash flow projections should EXCLUDE non-operating items)
Step 4: Sum: Operating value + Non-operating asset value = Total firm value
Step 5: Subtract debt (if using FCFF) → Equity value
Step 6: Divide by shares outstanding → Value per share
```

---

### Exam Tips

- **Don't double-count**: if non-operating assets generate cash flows that were included in the FCFF/FCFE forecast, don't add their value again separately
- **Always revalue to market**: book values of financial investments can be stale or based on historical cost
- **Common exam scenario**: a company holds "excess cash" significantly above operating needs — this should be added to the DCF-derived enterprise/equity value as a separate component
- **Land held for investment** (not used in operations) is a classic non-operating asset example
- This concept connects to **sum-of-the-parts (SOTP)** valuation discussed in corporate restructuring — same underlying logic of valuing components separately
